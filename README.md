# ghosttacoboy — menú de propuestas

12 propuestas de diseño (mobile-first) para la página personal de **ghosttacoboy**.
Una página `index.html` raíz que enlaza a cada variante en `v01..v12/`.

- `v01-recibo` … `v11-linkbio` — una filosofía de diseño por variante.
- `v12-generador` — generador contextual: arma un plan distinto según la hora, el día y la luna.

Borrador interno. Privado.
